# Algorithms and Data Structures

## Asymptotic growth notations

| Symbols         | Borders        | Growth |
| --------------- | -------------- | ------ |
| Θ (Big Theta)   | Top and Bottom | =      |
| O (Big O)       | Top            | <=     |
| o (Small O)     | Top            | <      |
| Ω (Big Omega)   | Bottom         | >=     |
| ω (Small Omega) | Bottom         | >      |
